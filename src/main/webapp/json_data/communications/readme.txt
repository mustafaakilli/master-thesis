In this folder all the JSON content related with Communications will be stored.

It has some default communications and settings.
The names of the following files MUST remain the SAME and MUST NOT be deleted..

-settingsDropdownElements.json
    This file has all the settings of the Dropdown boxes of the Communication tab.
    The settings of them can be configured using this file.
    More info see : http://cuppalabs.github.io/components/multiselectDropdown/

-Default.json
    - Contains all the possible selections of the entries in the Communication tab.

- And other default communication types.
    HTTP, AMQP, MQTT, DDS, STOMP, XMPP, CoAP
