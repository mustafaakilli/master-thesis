In this folder all the JSON content related with Environments will be stored.

It has default environment and settings.
The names of the following files MUST remain the SAME and MUST NOT be deleted.

-settingsDropdownElements.json
    This file has all the settings of the Dropdown boxes of the Environment and Analyser tab.
    The settings of them can be configured using this file.
    More info see : http://cuppalabs.github.io/components/multiselectDropdown/

-Default.json
    - Supported communications will be filled from Communications folder. It should be empty.
